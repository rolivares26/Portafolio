## Caso de estudio: Análisis de ventas y cumplimiento de metas en Power BI

### Breve descripción de la actividad o tarea
Este proyecto consistió en el desarrollo de un dashboard interactivo en Power BI para analizar el desempeño de ventas y el cumplimiento de metas en distintos periodos. Se trabajó con datos históricos, integrando información de ventas y metas para generar indicadores clave de rendimiento.

### Desafío principal
El principal desafío fue integrar correctamente distintas fuentes de datos (ventas y metas), establecer relaciones dentro del modelo de datos y calcular indicadores como el porcentaje de cumplimiento de metas de manera dinámica por mes.

### Solución propuesta
Se construyó un modelo de datos en Power BI que incluyó:
- Tabla de ventas
- Tabla de metas (archivo CSV)
- Tabla calendario para análisis temporal

Se crearon medidas en DAX para calcular:
- Total de ventas
- Meta total
- Cumplimiento de metas (%)

Se diseñó un dashboard con visualizaciones que permiten analizar el desempeño por periodo, identificar tendencias y detectar desviaciones respecto a las metas.
